# 💼 Blazor Portfolio

This is my personal portfolio built using **C#, Blazor, and Bootstrap**.

## 🚀 Features

* Multi-page routing (Home, Skills, Contact)
* Responsive design with Bootstrap
* Contact form with validation
* Clean and modern UI

## 🛠 Tech Stack

* C#
* Blazor Server
* ASP.NET Core
* Bootstrap

## 📂 Pages

* Home: Introduction and bio
* Skills: Technical skills categorized
* Contact: Form to reach out

## ▶️ Run Locally

```bash
dotnet run
```
